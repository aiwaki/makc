# Security Policy

## Supported Versions

Security fixes are handled on the latest tagged release and the `master`
branch.

## Reporting a Vulnerability

Please report sensitive issues through GitHub's private vulnerability reporting
flow:

https://github.com/aiwaki/makc/security/advisories/new

For non-sensitive hardening ideas or documentation gaps, open a regular GitHub
issue.

Please include:

- affected platform and backend;
- makc version or commit;
- minimal reproduction steps;
- expected and actual behavior.
